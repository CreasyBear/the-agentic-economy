# The Agentic Economy Brand Assets

Core files:

- `agentic-economy-mark.svg` — square AE monogram for avatar and compact placements.
- `agentic-economy-wordmark.svg` — wordmark for horizontal placements.
- `agentic-economy-logo.svg` — lockup with mark, wordmark, and tagline.
- `brandkit.json` — machine-readable brand system for agents, crawlers, and future tooling.

Color tokens:

- Ink: `#211f1a`
- Paper: `#eee9df`
- Accent: `#d24422`
- Moss: `#1f5548`

Typography:

- Display: DM Serif Display
- Body/UI: Bricolage Grotesque

Imagery:

- Quiet institutional field imagery.
- Editorial plates and atlas systems.
- No generic robot or synthetic AI imagery.

Voice:

- Quiet authority.
- Category ownership without over-explaining.
- Plain statements before terminology.
- Future-facing without foreboding.
